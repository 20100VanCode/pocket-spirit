# Community Creatures

This folder is for sharing your custom Pocket Spirit creatures with the community!

## How to Create a Creature

1. Copy `_template/` to a new folder with your creature's name
2. Implement all required methods from `CreatureBase`
3. Test on your device
4. Submit a PR!

## Guidelines

- Keep it family-friendly
- Each creature in its own folder
- Must implement all `CreatureBase` virtual methods
- Include a brief description in your header file
- Test before submitting
